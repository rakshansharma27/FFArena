        How to use this dynamic sitemap:


        Option 1 — Replace manually (easiest):

        Option 2 — Point Google here (best):

        Go to Google Search Console → Sitemaps → remove old sitemap → add:

        After saving a news post in admin dashboard, this URL pings Google:

        https://www.google.com/ping?sitemap=https://ffarena.live/sitemap-dynamic.html
